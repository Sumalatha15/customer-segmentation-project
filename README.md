# Customer Segmentation Project

## Overview
This project performs customer segmentation using K-Means clustering.

## Objective
To segment customers based on annual income and spending score.

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Dataset
Mall Customer Segmentation Dataset

## Methodology
1. Data loading
2. Data preprocessing
3. Data visualization
4. Elbow Method
5. K-Means Clustering
6. Segment visualization

## Results
Customers were grouped into different clusters based on purchasing behavior.

## Conclusion
Customer segmentation helps businesses understand customer behavior and improve marketing strategies.
